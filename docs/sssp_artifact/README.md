# Artifact: Reproducibility instructions

ASPLOS 2027 cycle 2 artifact-track submission. Paper draft: [`docs/manuscript/paper_v3.md`](../manuscript/paper_v3.md).

This directory contains everything needed to reproduce the paper's experimental claims from a clean checkout. Each numbered Finding (F17-F24 in `docs/manuscript/findings_log.md`) has a dedicated replication recipe in [`replication_recipes.md`](replication_recipes.md).

## TL;DR (one-liner reproduction)

```bash
git clone <repo>; cd gpu-sssp-certifying
bash scripts/install_rocm.sh        # AMD only (skip on NVIDIA hosts)
bash scripts/fetch_datasets.sh      # downloads ~5GB of graph data
mkdir build && cd build
# pick one:
cmake .. -DGPU_BACKEND=CUDA -DCMAKE_CUDA_ARCHITECTURES=86 \
         -DCMAKE_CUDA_COMPILER=/usr/local/cuda/bin/nvcc
cmake .. -DGPU_BACKEND=ROCM -DCMAKE_HIP_ARCHITECTURES=gfx942 \
         -DCMAKE_HIP_COMPILER=/opt/rocm/llvm/bin/amdclang++ \
         -DCMAKE_PREFIX_PATH=/opt/rocm
cmake .. -DGPU_BACKEND=NONE         # CPU-only (unit tests + CPU SSSP)
make -j$(nproc)

# §3.3 cross-vendor reproducibility matrix (run on each vendor):
bash ../scripts/run_strict_matrix.sh
# §5 controlled 2x2 (strict + relaxed):
cmake -B ../build_relaxed -DRELAX_ATOMICS=ON ...
bash ../scripts/run_e11_e12c.sh
# §6.5 Gunrock external library audit (NVIDIA only):
bash ../scripts/a10_e1_e2_remote.sh
# §7.2 Δ-sweep (per F24):
# (Δ values computed offline via the avg-edge-weight formula; the script logs them)
```

After all phases run, regenerate the data tables:

```bash
python3 ../scripts/sync_data_tables.py
```

Compare your generated `docs/plans/02_data_tables.md` against the committed version — every `d_hash` should match cell-by-cell (the strict-mode byte-equality claim is the artifact's canonical correctness witness).

## Hardware requirements

See [hardware_requirements.md](hardware_requirements.md). Summary:

- **NVIDIA**: any GPU with sm_60+ for FP32+packed atomic CAS path; sm_70+ recommended. CUDA 12.x. Tested on A10.
- **AMD**: any CDNA / RDNA-with-amdgpu GPU; ROCm 6.x or 7.x. Tested on MI300X VF.
- **CPU-only**: any x86_64 with C++20 compiler (tested on GCC 11+ and Clang 14+).

VRAM:
- ny_road / web_google: < 1 GB
- usa_road / livejournal: ~4-8 GB
- RMAT-22 to RMAT-25: ~2-25 GB
- RMAT-26 (not tested in submission): ~50+ GB → out of scope here

## Build matrix

| Backend | Compiler | Notes |
|---|---|---|
| `-DGPU_BACKEND=NONE` | g++ / clang++ | CPU-only build for unit tests |
| `-DGPU_BACKEND=CUDA` | nvcc + `hip_cuda_compat.h` | Compiles `delta_stepping.hip` as CUDA via the LANGUAGE CUDA property |
| `-DGPU_BACKEND=ROCM` | `amdclang++` | Native HIP compilation; targets gfx942 by default (override with `-DCMAKE_HIP_ARCHITECTURES=…`) |
